<h2>導入方法</h2>
Japaneseフォルダを、DungeonsAndLesbians-1.2-win/game/tlフォルダの中においてください。<br>
次回起動時から日本語化した状態で始まります。<br>
英語に戻したいときは導入したJapaneseフォルダを削除してください。<br>
